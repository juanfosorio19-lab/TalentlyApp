# Workflow: Nueva Vista en Talently

1. Identificar qué vista se va a crear (swipe, chat, perfil, 
   onboarding, filtros, configuración)
2. Usar el Stitch MCP server para generar el diseño visual 
   de esa vista con el estilo de Talently
3. Revisar el diseño generado y ajustar si es necesario
4. Implementar el HTML/CSS/JS basándose EXACTAMENTE en el 
   diseño de Stitch aprobado
5. Verificar consistencia visual con el resto de las vistas 
   existentes de Talently
6. Confirmar que funciona en mobile (mobile-first)
