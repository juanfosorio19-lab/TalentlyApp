## Verificaciones DB (una sola query consolidada)

```sql
SELECT
  (SELECT COUNT(*) FROM pg_class WHERE relnamespace='public'::regnamespace
   AND relkind='r' AND relrowsecurity=false)        AS rls_off,
  -- policies viejas permisivas
  (SELECT COUNT(*) FROM pg_policies WHERE schemaname='storage'
   AND policyname IN ('Subida Permitida','Borrado Permitido',
                      'Authenticated users can upload videos')) AS old_storage_policies,
  -- RPCs SECURITY DEFINER críticas
  (SELECT COUNT(*) FROM pg_proc WHERE pronamespace='public'::regnamespace
   AND proname IN ('delete_account','increment_stat','log_daily_activity')
   AND prosecdef=true)                              AS rpcs_security_definer,
  -- columnas de metadata pobladas
  (SELECT COUNT(*) FROM public.company_positions
   WHERE icon IS NULL)                              AS positions_no_icon,
  (SELECT COUNT(*) FROM public.tech_stack
   WHERE abbreviation IS NULL)                      AS tech_no_abbrev,
  (SELECT COUNT(*) FROM public.profiles
   WHERE modality IN ('remote','hybrid','onsite'))  AS profiles_lowercase_modality;
```

## Sección 12 — Routing Guards ✅ (con RoleGate)

`App.jsx` ahora tiene los **4 layers de defensa**:

```
Line 4: import PrivateRoute        ← capa 1: autenticación
Line 5: import RoleRedirect        ← layout para /dashboard
Line 6: import OnboardingGate      ← capa 2: profile + onboarding
Line 7: import RoleGate            ← capa 3: rol vs ruta ⭐ nuevo

Line 76: <Route element={<PrivateRoute />}>
Line 79:   <Route path="/dashboard" element={<RoleRedirect />} />
Line 86:   <Route element={<OnboardingGate />}>
Line 88:     <Route element={<RoleGate type="candidate" />}>
            ...  /app/* routes
Line 104:    <Route element={<RoleGate type="company" />}>
             ...  /company/* routes
```

Todas las rutas privadas pasan por 3 guards en cadena. Si falla cualquiera, redirect correcto:
- PrivateRoute falla → `/login`
- OnboardingGate falla → `/onboarding/<tipo>`
- RoleGate falla → `/dashboard` (RoleRedirect lo manda al rol correcto)
